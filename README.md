# xrwvm-fullstack_developer_capstone

## Project Name
Full Stack Developer Capstone Project

## Description
This project is a full-stack web application developed as part of the Full Stack Developer Capstone.  
It allows users to browse car dealerships, view dealership details, and read or submit reviews.

The application uses:
- **Django** for the backend services and APIs
- **React** for the frontend user interface
- **MongoDB / Cloudant** for storing dealership and review data

The goal of the project is to demonstrate full-stack development skills including backend development, frontend development, API integration, and deployment.
